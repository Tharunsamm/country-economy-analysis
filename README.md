# Country Economy Analysis

This project analyzes country-level economic data using Pandas and visualization libraries.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Country-level data including:
- Birth Rate
- Internet Users
- Income Group
